# Lesson 1.3 – Mina länkar (tabell)

Detta steg introducerar en ny sida:

- `links.html`

Här arbetar du med en **HTML-tabell** för att strukturera information i rader och kolumner.

Syftet är att:
- förstå hur tabeller byggs upp
- se hur rader (`<tr>`) och celler (`<td>`, `<th>`) fungerar
- kombinera tabeller med länkar (`<a>`)
